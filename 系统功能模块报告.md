## Core Functional Modules

### 1. Natural Language to SQL Query

**Function Description**: Users input questions in Chinese or English, and the system automatically understands the intent, retrieves database Schema, generates and executes SQL queries.

![image-20251116230219771](../Library/Application Support/typora-user-images/image-20251116230219771.png)

---

### 2. Data Visualization

**Function Description**: Automatically identifies chart requirements and generates various types of interactive charts including bar charts, line charts, area charts, pie charts, and more.

![image-20251116233647054](../Library/Application Support/typora-user-images/image-20251116233647054.png)

---

### 3. Data Export Function

**Function Description**: Supports exporting query results and charts. Users can export data tables, charts, or generate comprehensive analysis reports.

**Export Types**:

- **Chart PNG**: Export interactive charts as high-resolution PNG images
- **Data Export**: Export query results as CSV or Excel files with proper formatting

![image-20251116233439315](../Library/Application Support/typora-user-images/image-20251116233439315.png)

---

### 6. Multi-turn Dialogue and Context Memory

**Function Description**: Supports continuous dialogue, where the system remembers previous query results and handles context-related questions such as "continue" or "only show last results".
